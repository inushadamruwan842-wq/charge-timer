# ⏱️ Charge Timer

This repository contains the core code for the Charge Timer project.

⏳ Adjustable timing: 3 minutes to 5.5 hours. PowerCut protection included. Used Arduino and a LCD module

🏆 Winner – Circuito 21 New Invention Competition (2021) ; Got a opportunity to post an article in "DINAMINA" newspaper DHWARA section
